# RAG Complaint Chatbot

This project implements a Retrieval-Augmented Generation (RAG) chatbot for intelligent complaint analysis.

## Structure
- `data/`: Contains raw and processed data.
- `vector_store/`: Stores the persisted FAISS/ChromaDB index.
- `notebooks/`: Jupyter notebooks for experimentation.
- `src/`: Source code for the application.
- `app.py`: Main entry point for the Gradio/Streamlit interface.
